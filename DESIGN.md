# 知识库查询前端 — 赛博朋克风格

> **设计风格**：Active Theory (WebGL诗人) + Ash Thorp (赛博诗意) + Territory Studio (FUI)
> **技术栈**：HTML + CSS + Canvas 2D + Vanilla JS
> **日期**：2026-05-03
> **状态**：Phase 1 完成 ✅

---

## 设计哲学

> 「让数据流动可见化」

搜索不是输入文字——是**在数据的海洋中发射一道光**。

## 核心视觉元素

### 粒子系统
- 深蓝/黑色背景中漂浮的粒子云
- 粒子颜色：青色 (#00f0ff)、品红 (#ff006e)、蓝紫 (#720982)、琥珀 (#ffbe0b)
- 粒子随鼠标移动产生引力场效果
- 搜索时粒子向搜索框汇聚

### 搜索界面
- 半透明 HUD 风格面板 (glassmorphism)
- 搜索框带发光边框 + 呼吸灯动画
- 科幻风格的字体排印 (Orbitron / Rajdhani)

### 搜索结果
- 3D 倾斜卡片 (CSS perspective + rotateX/Y)
- 悬浮时粒子在卡片周围聚集
- 霓虹辉光边框

### 色彩方案
```css
--deep-space: #0a0a1a;       /* 深空背景 */
--nebula: #0d1b2a;           /* 星云渐变 */
--cyan-glow: #00f0ff;        /* 主霓虹青 */
--magenta-glow: #ff006e;     /* 辅助霓虹粉 */
--purple-glow: #720982;      /* 紫色点缀 */
--amber-glow: #ffbe0b;       /* 琥珀强调 */
--text-primary: #e0e0ff;     /* 主文字 */
--text-secondary: #8888aa;   /* 次文字 */
```

## 📝 设计决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 粒子渲染 | Canvas 2D（当前）→ Three.js（升级） | 先快速出效果，后续 GPU 升级 |
| 字体 | Orbitron + Rajdhani + Share Tech Mono | 科幻感 + 可读性平衡 |
| 搜索方式 | 客户端 mock（当前）→ 后端 API | 快速原型验证 |
| 卡片交互 | 3D 倾斜 + 光追渐变 | Ash Thorp 标志性手法 |
| 色彩 | 青+品红双主色 | 经典赛博朋克配色 |
