# NEXUS Knowledge Base — 产品文档

> **产品名称**：NEXUS Knowledge Base（知识枢纽）  
> **产品定位**：赛博朋克风格的沉浸式知识库检索系统  
> **文档版本**：v1.0  
> **更新日期**：2026-04-19  

---

## 一、产品概述

### 1.1 一句话描述

NEXUS 是一个**沉浸式赛博朋克风格**的知识库检索前端界面，将枯燥的文档搜索转化为**宇宙探索体验**——用户在粒子星云中航行，以飞船穿越知识星球，获取集体智慧。

### 1.2 设计哲学

> 「让数据流动可见化」

搜索不是输入文字——是**在数据的海洋中发射一道光**。

### 1.3 目标用户

| 用户角色 | 核心需求 | 使用场景 |
|---------|---------|---------|
| 工程师/开发者 | 快速检索技术文档、最佳实践 | 编码过程中遇到技术问题 |
| 技术管理者 | 了解团队知识资产分布 | 技术选型、架构评审 |
| 知识管理者 | 维护和更新知识库配置 | LLM 配置、搜索策略调优 |

### 1.4 竞品差异化

| 维度 | 传统搜索工具 | NEXUS |
|------|------------|-------|
| 视觉风格 | 白底黑字，功能优先 | 赛博朋克，沉浸式体验 |
| 背景效果 | 静态/无 | Three.js 3D 粒子系统（3000+ 粒子） |
| 交互模式 | 列表式结果 | 星球宇宙 + 传统卡片双模式 |
| 搜索增强 | 关键词匹配 | LLM AI 总结 + 联网补充 |
| 系统配置 | 无/后端配置 | 前端管理面板，实时生效 |

## 二、核心功能

### 2.1 知识搜索

**核心能力**：
- **关键词搜索**：支持多词匹配，标题/摘要/标签/分类多权重评分
- **实时搜索**：输入 2+ 字符自动触发（500ms 防抖）
- **快捷标签**：8 个预设技术标签（React/TypeScript/微服务/数据库/Docker/AI/安全/GraphQL）
- **排序切换**：相关度 / 最新 / 热度

**搜索结果展示**：
- 传统模式：3D 倾斜卡片，含标题、摘要、标签、来源、匹配度
- 探索模式：星球宇宙，每个结果是一颗自转星球

### 2.2 文档详情

**侧边抽屉面板**，点击结果卡片后滑出，包含：
- 文档元信息（分类、ID、来源、匹配度、更新时间）
- 标签云
- 摘要预览
- **完整正文**（Markdown → HTML 渲染，基于 `marked.js`）
- 相关文档推荐（同分类/同标签自动关联）
- **原文加载**：通过 API 获取完整文档内容
- **收藏功能**：localStorage 持久化

### 2.3 宇宙探索模式

**🚀 核心亮点功能**——将搜索结果转化为可交互的宇宙空间：

#### 飞船系统
- **SVG 矢量飞船**：精细设计的太空飞船（机身渐变、发光驾驶舱、引擎火焰）
- **鼠标跟随**：飞船平滑插值跟随鼠标移动（lerp factor 0.03）
- **自动朝向**：船头自动旋转朝向鼠标方向
- **引擎推进粒子**：飞船尾部喷射青色/品红粒子
- **尾迹系统**：飞行时留下发光轨迹
- **加速飞行**：搜索时飞船加速，产生流光粒子效果

#### 星球系统
- **3D 自转星球**：搜索结果以星球形式展示，绕 Y 轴 360° 自转
- **差异化旋转**：每个星球不同速度（8-18s）和方向（正反交替）
- **线框模式**：默认显示经纬线线框 + 半透明
- **Hover 交互**：
  - 星球显示真实颜色（8 色方案：蓝/粉/绿/金/紫/青/橙/薰衣草）
  - 旋转加速 4 倍
  - 弹出信息气泡（标题/摘要/标签/分类/匹配度）
- **光环系统**：随机星球带有倾斜光环（类似土星）

#### 飞船开火
- **点击空白区域**：飞船向点击方向发射子弹
- **双枪系统**：左枪（青色）+ 右枪（品红），各有发光拖尾
- **震动反馈**：开火时飞船微震动
- **可配置开关**：通过 localStorage 控制

#### 粒子背景
- **Three.js 3D 粒子云**：3000 个粒子，GPU 渲染
- **4 色分布**：青色 45% / 品红 25% / 蓝紫 20% / 琥珀 10%
- **鼠标引力场**：8 单位半径内粒子被鼠标吸引
- **搜索汇聚**：搜索框聚焦时粒子向中心汇聚
- **粒子连线**：近距离粒子间绘制发光连线
- **探索模式加速**：粒子速度提升 4 倍

### 2.4 LLM AI 增强

**可选功能**（管理面板配置）：
- **AI 总结**：搜索结果由 LLM 生成 Markdown 格式的综合摘要
- **联网补充**：自动从互联网搜索补充相关知识
- **Fallback 机制**：LLM 不可用时自动降级为纯文本摘要

### 2.5 系统管理面板

**⚙ SETTINGS 侧边抽屉**，提供运行时配置：

| 配置模块 | 选项 | 说明 |
|---------|------|------|
| LLM 配置 | Backend / Model / Base URL / API Key | 支持 OpenAI / Ollama / 自定义 |
| 搜索默认值 | AI 总结开关 / 联网补充开关 / top_k / 去重模式 | 控制搜索行为 |
| 服务器配置 | CORS 来源 | 跨域白名单 |

配置通过 REST API 保存到后端，重启不丢失。

## 四、交互流程

### 4.1 搜索流程

```
用户访问页面
    │
    ▼
看到粒子星云背景 + HUD 导航
    │
    ▼
输入关键词 / 点击快捷标签 / 回车
    │
    ▼
搜索框发光 + 粒子汇聚 + 飞船加速
    │
    ▼
显示加载动画（双环旋转 + "SEARCHING KNOWLEDGE MATRIX..."）
    │
    ▼
后端 API 处理（可选 LLM 总结 + 联网补充）
    │
    ▼
结果展示
    ├── 普通模式：3D 卡片网格升起
    └── 探索模式：星球宇宙 + 飞船跟随
    │
    ▼
点击卡片 → 详情面板滑出
    │
    ▼
查看内容 / 加载原文 / 收藏 / 查看相关文档
```

### 4.2 探索模式交互

```
点击 🚀 EXPLORE MODE
    │
    ▼
粒子加速扩散 + 飞船出现 + 飞船跟随鼠标
    │
    ▼
执行搜索 → 结果以星球形式展示
    │
    ▼
鼠标移入星球 → 显示真实颜色 + 旋转加速 + 信息气泡
    │
    ▼
点击星球 → 详情面板滑出
    │
    ▼
点击空白区域 → 飞船开火（双子弹射击）
    │
    ▼
点击 🚀 EXPLORE ON → 退出探索模式
```

### 4.3 管理面板流程

```
点击 ⚙ SETTINGS
    │
    ▼
管理面板从右侧滑出
    │
    ▼
配置 LLM / 搜索策略 / 服务器
    │
    ▼
点击「保存配置」→ API 保存 → Toast 提示成功
    │
    └── 或点击「重置默认」→ 确认 → 恢复出厂配置
```

## 五、技术栈总览

| 层次 | 技术 | 用途 |
|------|------|------|
| 结构 | HTML5 | 语义化页面结构 |
| 样式 | CSS3 (Houdini @property) | 赛博朋克视觉 + 动画 |
| 3D 渲染 | Three.js r160 | 粒子系统 GPU 渲染 |
| 逻辑 | Vanilla JavaScript | 无框架，原生实现 |
| 文本渲染 | Marked.js | Markdown → HTML |
| 字体 | Google Fonts | Orbitron / Rajdhani / Share Tech Mono |
| 后端 | knowledge-search-tool-server (Python/FastAPI) | 搜索 API + LLM |
| 存储 | localStorage | 用户偏好 + 收藏 |

## 六、配置与部署

### 6.1 前端配置

```javascript
// 后端 API 地址（通过环境变量或全局变量配置）
window.KB_API_BASE = 'http://localhost:8000';  // 默认值

// 探索模式开关（localStorage 持久化）
localStorage.setItem('nexus-explore-mode', '1');

// 飞船开火开关
localStorage.setItem('nexus-ship-fire', 'true');  // 默认开启
```

### 6.2 后端 API 端点

| 端点 | 方法 | 用途 |
|------|------|------|
| `/api/search` | POST | 知识搜索（含 LLM 总结） |
| `/api/document/{path}` | GET | 获取完整文档内容 |
| `/api/admin/config` | GET/PUT | 管理配置读写 |
| `/api/admin/config/reset` | POST | 重置配置 |

### 6.3 部署方式

```bash
# 静态文件部署（任意 HTTP 服务器）
# 仅需将 webDesign/ 目录部署到 Web 服务器

# 示例：使用 Python 快速启动
cd webDesign
python3 -m http.server 3000

# 示例：使用 Nginx
# 将目录配置为静态资源根目录
```
